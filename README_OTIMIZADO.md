## 🚀 **Início Rápido**

### **Pré-requisitos:**
- Node.js 18+
- MySQL 8.0+
- npm ou yarn

### **Instalação:**

```bash
# 1. Clone o repositório
git clone https://github.com/edgthiago/voePapel.git
cd voePapel

# 2. Configure o backend
cd backend
npm install
cp .env.example .env
# Configure suas variáveis de ambiente no .env

# 3. Configure o frontend
cd ../frontend
npm install

# 4. Execute em desenvolvimento
# Terminal 1 - Backend
cd backend && npm run dev

# Terminal 2 - Frontend  
cd frontend && npm run dev
```

### **URLs de Acesso:**
- **Frontend:** http://localhost:3000
- **Backend API:** http://localhost:30011
- **Documentação:** http://localhost:30011/api/docs
